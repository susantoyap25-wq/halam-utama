# Portofolio Pribadi Susanto Yap

Website portofolio pribadi yang modern, profesional, dan responsif untuk menampilkan profil, keahlian, dan pengalaman Susanto Yap.

## Fitur

- **Desain Modern**: Menggunakan Tailwind CSS untuk styling yang clean dan profesional
- **Responsif**: Tampil optimal di desktop dan mobile
- **Animasi Halus**: Transisi dan efek hover untuk pengalaman pengguna yang baik
- **Section Lengkap**: Hero, About, Skills, Experience, Projects, Values, Testimonials, Contact, Footer

## Struktur File

```
/
├── index.html          # File HTML utama
├── css/
│   └── style.css       # Custom styles
├── js/
│   └── script.js       # JavaScript untuk interaktivitas
└── assets/
    └── images/         # Folder untuk gambar (gunakan placeholder saat ini)
```

## Cara Menggunakan

1. Buka `index.html` di browser web
2. Website akan langsung dapat dilihat dan digunakan

## Kustomisasi

- **Nama dan Informasi**: Sesuaikan teks di `index.html` untuk menampilkan nama, profil, dan pengalaman Anda
- **Konten**: Sesuaikan teks di setiap section sesuai dengan profil Anda
- **Gambar**: Ganti placeholder images dengan foto dan gambar project Anda
- **Warna**: Sesuaikan palet warna di Tailwind classes
- **Skills**: Ubah persentase dan warna progress bar di section skills

## Teknologi

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (ES6+)
- Font Awesome (untuk ikon)

## Browser Support

- Chrome (recommended)
- Firefox
- Safari
- Edge

## Catatan

- Website ini menggunakan CDN untuk Tailwind CSS dan Font Awesome
- Placeholder images menggunakan via.placeholder.com
- Untuk production, sebaiknya host gambar sendiri dan minify CSS/JS
